This folder contains all of the files relevant to performing geometry optimizations in OpenMolcas

tbhp_s0_sa4_casscf_opt.input is the input file. It is the S0 geometry optimization of TBHP at SA(4)-CASSCF(6e,4o)/cc-pVDZ level of theory

tbhp_s0_sa4_casscf_opt.log is the output file 

The tbhp_s0_sa4_casscf_opt.Opt.xyz is the final optmized geometry produced from the calculation
