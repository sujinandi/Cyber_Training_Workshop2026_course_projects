This folder contains inputs and outputs of calculations done with PySCF and Prism

core_mr_adc: Calculation of X-ray photoelectron spectra, oxygen K-edge, of furan. Uses a CASSCF(6,6) with a MR-ADC(2)-x calculation. Produced XPS spectra for the ground state and two excited states. Based on the result, it seems that there are a lack of spectral features showing that for the calculation, there is no significant oxygen 1s ionization that occurs within the 540 to 550 eV range. I included the associated plots and data/txt files

eom_ccsd: Calculation of naphthalene using EOM-CCSD calculation with frozen core and density fitting. The ten lowest energy singlet states were calculated
