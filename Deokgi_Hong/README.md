# Photodissociation Dynamics of CH<sub>4</sub> Using PySpawn/OpenMolcas
CyberTraining 2026 Workshop Project
Author: Deokgi Hong

# Overview
This project applies the AIMS (ab initio multiple spawning) method, using PySpawn/OpenMolcas, to CH<sub>4</sub> photodissociation dynamics. In this work, electronic population and branching ratios from 32 simulations were analyzed, confirming that trajectories following different reaction pathways can be observed within a single simulation.
