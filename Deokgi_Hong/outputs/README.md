# Outputs
### {state}_pop.dat
Change of nuclear population over time (in fs)

### E_{state}.dat
Change of potential, kinetic, and total energy over time (in a.u.) of each state

### Labels.dat
Labels of each state

### N.dat
Change in electronic state over time (in a.u.)

### product_pop.dat
Change in population of different structures over time (in fs)

Data is written in the order of time, CH<sub>4</sub>, CH<sub>3</sub> + H, CH<sub>2</sub> + H<sub>2</sub>, CH<sub>2</sub> + 2H, CH + H<sub>2</sub> + H

### traj_class.dat
Structure of each trajectories
