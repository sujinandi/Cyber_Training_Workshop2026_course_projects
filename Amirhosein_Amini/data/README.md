# Input data

- `2H2Pc_C60.xyz`: published 176-atom donor-acceptor geometry.
- `fragments.json`: atom ranges used for donor and C60 projectors.
- `production_starts/`: exact ten starting structures used in production; `manifest.csv` records their source frames.
- `experiment_shg_digitized.csv`: approximate digitization of the published SHG trace, not original raw experimental data.
