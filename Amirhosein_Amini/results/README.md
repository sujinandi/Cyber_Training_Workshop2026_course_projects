# Final outputs

This directory contains the compact, submission-ready numerical outputs and the original figures generated by the completed coherent-dynamics and Libra analyses. No TENSO result is retained.

## Numerical results

| File | Contents |
|---|---|
| `current_results.json` | Complete headline metrics used in the README and report |
| `fssh_summary.csv` | Endpoint and error summary for plain FSSH, Boltzmann-rescaled FSSH, and experiment |
| `coherent_summary.json` | Full/reduced coherent validation, coherence, and current diagnostics |
| `coherent_ensemble.csv` | Time-resolved full and reduced coherent ensemble means and 95% confidence intervals |
| `donor_acceptor_channel_flux.csv` | Pair-resolved signed, positive, and absolute integrated fluxes |
| `libra_experiment_comparison.csv` | Time-resolved coherent, active-surface, and hybrid Libra observables |
| `plain_boltzmann_experiment.csv` | Time-resolved plain/Boltzmann/experiment comparison with uncertainty bands |
| `libra_comparison_metrics.txt` | Human-readable plain Libra comparison metrics |
| `plain_boltzmann_metrics.txt` | Human-readable detailed-balance sensitivity metrics |

## Original figures

The `figures/coherent/` and `figures/libra/` directories contain the original PDF and PNG files written by the final analysis scripts:

- full ten-state versus reduced 4D+3A coherent propagation;
- donor-acceptor coherence;
- net donor-to-acceptor current;
- dominant transfer channels;
- the ten-trajectory Libra hybrid ensemble;
- Libra observable decomposition;
- Libra versus digitized experiment;
- plain versus Boltzmann-rescaled CPA-FSSH;
- the isolated Boltzmann-rescaling effect.

The multi-gigabyte raw trajectories and frame-resolved arrays are computational intermediates and remain excluded. They are regenerated from the fixed structures, seeds, scripts, and Slurm templates retained in the repository.
